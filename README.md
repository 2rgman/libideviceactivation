# libideviceactivation

## About

A library to manage the activation process of Apple iOS devices.

## Requirements

Development ackages of:
* libcurl
* libxml2
* libplist
* libimobiledevice

Software:
* usbmuxd
* make
* autoheader
* automake
* autoconf
* libtool
* pkg-config
* gcc

## Installation

To compile run:
```bash
./autogen.sh
make
sudo make install
```
